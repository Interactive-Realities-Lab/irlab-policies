## Current and Recent Courses

### Iowa State University

**COMS 4740/5740: Introduction to Machine Learning** (Spring 2025)  
Introduction to concepts, tools and techniques of machine learning for applications. Selected machine learning algorithms in practical data mining tasks such as classification, regression, and clustering (association rules, decision trees, linear models, Bayesian learning, support vector machines, artificial neural networks, instance-based learning, probabilistic graphical models, ensemble learning, and clustering algorithms).

**COMS 3630: Introduction to Database Management Systems** (Fall 2024)  
Data models and database design using entity relationship model. Database implementation with relational and graph database management systems (DBMS) and query languages. DBMS internals including basic concepts of database storage, indexing, query optimization, and concurrent control. Database application development.

### University of North Carolina at Greensboro

**CSC471: Principles of Database Systems** (2020-2023)  
Contemporary database systems with emphasis on query processing, design, and implementation of applications in relational (SQL) databases. Introduction to other database models such as XML, object-oriented, and deductive.

**CSC495/693: Virtual Reality** (2020-2023)  
Comprehensive overview of state-of-the-art in virtual reality technologies and 3D user interfaces. Topics include display systems, motion tracking, spatial interaction techniques, and human factors. Hands-on programming assignments using Oculus Quest headsets. Theoretical and practical knowledge evaluated through quizzes and course projects.

**CSC462/662: Principles of Operating Systems** (2022, 2024)  
Techniques and strategies used in operating system design and implementation—managing processes, input/output, memory, scheduling, file systems, and protection.

**CSC442/642: Human-Computer Interface Development** (2020-2024)  
Survey of concepts and techniques for human-computer interface development. Topics include user-centered design, user interface programming, and usability evaluation.

**IAF603: Preparing Data for Analytics** (Fall 2019)  
Current approaches, techniques and best practices for collecting, cleaning and normalizing data, processing, storing, managing, securing and preparing structured and unstructured big data sets for analytics.
